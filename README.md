A collection of scripts written in [Lean4](https://lean-lang.org).

They are meant to be self-contained and only use Mathlib when necessary.

The only requirement is a working installation of Lean4.
I recommend installing [elan](https://github.com/leanprover/elan) and using `lean` from the command line.
Assuming the command line is familiar with Lean, the scripts can be run as follows:

```sh
lean scripts/example.lean
```
